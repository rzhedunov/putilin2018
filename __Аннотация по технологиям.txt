
Краткая сводка по используемым технологиям:

Использована ASP.Net MVC
На страницах добавлен javascript для заполнения полей и работы с датами в полях и Datapicker'е
Доступ реализован встроенный с хранением данных в MS SQL
Используется Entity Framework (DB first) с хранимыми процедурами и табличными функциями (нетабличные он не берет)
В БД созданы объекты логики сервера, в т.ч., для получения большого сводного отчета.
Сайт выложен на внешний хостинг parking.ru (без панели детальной настройки сервера)
Для управления версиями использован Github.
Тесты изначально предполагались, но не использованы.
